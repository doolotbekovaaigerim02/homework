git add - добавляет содержимое католога в индекс(staging area) для последующего коммита.
git status - показывает состояние файлов в рабочем катологе и индексе: изменение файлов,какие файлы не добавлены в коммит.
git diff - показывает изменения между двумя любыми Git деревьями.(Просмотр изменений до коммита)
dit difftool - запускает внешнюю утилиту для показа различий в двух деревьях.
git commit - берет все данные ,добавленые в индекс  с помощью git add, и сохраняет их в слепок во внутренней базе данных,а затем сдвигает указатель текущей ветки на этот слепок.
git reset - изпользуется для отмены изменений для всех подготовленных файлов, путь к ним не надо указывать.
git rm - изпользуется для удаления файлов из индекса и рабочей копии.
git mv - для перемещения файлов.(переименование файлов).
git clean - для удаления мусора из рабочего католога.
git config - кеширование учетных данных.
git init - создать пустой репозиторий Git или вновь инициализировать существующий можно параметром init.
git log - для просмотра( историй) изменений, внесенных  в репозиторий.
git checkout - чтобы восстановить файлы рабочего дерева, не подготовленные к коммиту.
git commit --amend - внести изменения в последний коммит.
git revert -для откатывания последнего коммита(разница между revert и reset: revert отменяет изменения записанные только одним коммитом.она не откатывает проект к юолее раннему состояние,удаляя все последующие коммиты ,как делает это reset) 
git branch - сщздать новую ветку можно с помощью параметра branch,указав имя ветки.
git remote - Просмотр удаленных URL-адресов.
git remote show - получить дополнительные сведения об удаленном репозитории.
git push - отправлять изменения в удаленный репозиторий можно  параметром push.
git pull - для загрузки изменений из удаленного репозитория используется параметр pull.
git merge - слияние удалееного репозитория с локальным выпольняется параметром merge с указанием имени удалееного репозитория.
git ls-files - показывает информацию о файлах в индексе
git rm - позволяют удалять отдельные файлы или группы файлов(удаляет отслеживаемые файлы из раздела проиндексированных файлов Git)
